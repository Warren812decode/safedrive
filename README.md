# Safedrive

## Setup

```bash
cd frontend && npm install
cd ../backend && npm install
cd .. && npm install
```

## Development
Run both frontend and backend together:
```bash
npm run dev
```

- Frontend: http://localhost:5173
- Backend: http://localhost:5000

## Build frontend
```bash
npm run build
```

## Start backend only
```bash
npm start
```
